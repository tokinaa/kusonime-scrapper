## Laravel x Vue JS
My first SPA project with Laravel x Vue, in this project i choose scrapping for learning.

## How to install
- Clone this repository
- `npm install` for installing node library
- `npm run watch`
- `php artisan serve`
- Type this in your website `127.0.0.1:8000` and enjoy !
